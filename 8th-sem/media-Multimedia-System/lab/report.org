* Lab Work
** Create a 3D table in Blender
A simple table can be created in Blender by starting with basic cubes and reshaping them into parts like the tabletop and legs. This approach helps beginners understand how most 3D models start from simple geometry and are refined through basic transformations, giving a clear and practical introduction to working with objects in a three dimensional space.
#+LATEX:\newline

**Steps:**
1) Create a new project in Blender
2) Select the cube
3) Flatten and lengthen it using the Scale tool (S) to create the top
   #+attr_latex: :height 220px
   [[file:img/1.png]]
4) Add a new cube to make the table leg
5) Scale it on the Z-axis to be tall
6) Move it to below the table top using the move tool (G)
7) Duplicate it three times using Shift+D
8) Move the other table legs in the appropriate position
9) Add materials if needed to color it

** Create a 3D glass in Blender
A glass can be created by starting with a basic cylinder, then extruding and scaling its faces to create the rim, and hollow interior. Applying a glass material helps us understand shaders and how transparency works, while adding lighting shows how reflections and refraction affect the final look. 
#+LATEX:\newline

**Steps:**
1. Launch Blender and create a fresh scene.
2. Remove the default cube by selecting it and pressing X to delete.
3. Build the base plate by adding a Cylinder from the Mesh menu.
4. Flatten it by scaling along the Z axis, then widen it to form a round plate.
 #+attr_latex: :height 220px
 [[file:img/3.png]]

5. Model the glass by adding another Cylinder and stretching it upward to form the main body.
6. Enter Edit Mode, select the top face, extrude it slightly, and scale inward to create the rim.
7. Extrude downward to form the hollow interior.
8. Move the glass upward along the Z axis until it sits correctly on the plate.
 #+attr_latex: :height 220px
 [[file:img/4.png]]

9. Switch back to Object Mode.
10. Set up the glass material. Open the Material panel with the glass selected and add a new material.
11. Choose Glass BSDF as the shader and set the IOR to about 1.45.
12. Lower Roughness if you want a clearer appearance.
13. Add lighting and a surface. Insert an Area Light above the glass and increase its intensity for reflections.
14. Place a Plane underneath to serve as the table or ground surface.
